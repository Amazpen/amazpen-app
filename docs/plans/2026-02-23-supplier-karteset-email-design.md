# עיצוב: בקשת כרטסת אוטומטית לספקים

**תאריך:** 2026-02-23

## סיכום

הוספת שדה מייל לכרטיס ספק + toggle לבקשת כרטסת אוטומטית כל 2 לחודש.
n8n workflow שולח את המיילים מכתובת מערכת עם CC לבעל העסק.

## שינויים נדרשים

### 1. DB - טבלת suppliers

- עמודה `email` — כבר קיימת, לא צריך שינוי
- עמודה חדשה: `request_karteset` (boolean, default false)

### 2. ממשק - טופס ספק (suppliers/page.tsx)

**שדות חדשים בטופס הוספה/עריכה:**
- שדה "כתובת מייל" — input type=email
- Toggle "שלח בקשת כרטסת כל 2 לחודש" — מופיע רק כשיש מייל

**בתצוגת כרטיס ספק:**
- אייקון מעטפה ליד שם ספק עם request_karteset=true

### 3. n8n Workflow - שליחת כרטסת

**Trigger:** Schedule — כל 2 לחודש (לדוגמה: ב-1 וב-15)
**שלבים:**
1. שליפת ספקים: `request_karteset=true AND email IS NOT NULL AND is_active=true`
2. לכל ספק — שליפת פרטי עסק + מייל בעל העסק
3. שליחת מייל מכתובת מערכת (noreply@brainboxai.io) עם CC לבעל העסק
4. תוכן: בקשה מנומסת לכרטסת, שם עסק, תקופה

### 4. תוכן המייל

```
נושא: בקשת כרטסת - [שם העסק]

שלום רב,

אנו פונים אליכם בבקשה לקבלת כרטסת עדכנית עבור [שם העסק].
נודה לקבלת הכרטסת לתקופה [חודש/שנה נוכחיים] לכתובת מייל זו.

בתודה מראש,
[שם העסק]
```

## החלטות

- מייל נשלח מכתובת מערכת קבועה (לא מייל בעל העסק)
- בעל העסק מכותב (CC) בכל מייל
- Toggle בכרטיס ספק מסמן אם הספק נכלל בשליחה
- ספקים ללא מייל — Toggle לא מופיע
